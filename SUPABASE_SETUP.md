# Guía para obtener la SERVICE_ROLE_KEY de Supabase

1. Ve a tu proyecto en [Supabase Dashboard](https://supabase.com/dashboard)
2. Selecciona el proyecto **"MyBooking"** (o el nombre de tu proyecto)
3. En el menú lateral, haz clic en **Settings** (Configuración)
4. En la sección **API**, busca **Project API keys**
5. Verás una clave llamada **service_role secret** (no la ANON_KEY)
6. Haz clic para revelar la clave y cópiala

**Nota:** Esta clave tiene permisos de administrador, no la compartas públicamente.

---

## Variables de entorno a configurar

Una vez tengas la SERVICE_ROLE_KEY, por favor compártela para completar la configuración.
